# CodeCT - 代码统计工具

一个VS Code扩展，用于跟踪和统计您的编程活动数据。

## 功能特性

- 记录详细的编码活动数据
- 按项目、语言分类统计
- 显示最近7天的详细编码时间段
- 超过7天的数据自动汇总统计
- 支持导出JSON、CSV、HTML格式报告
- 美观的HTML可视化报告
- 实时统计输入字符数
- 记录删除的字符数
- 跟踪代码行变更
- 统计编码时长
- 支持多项目管理
- 导出统计报告(JSON/CSV/HTML)

## 安装方法
1. 下载最新版本VSIX文件
2. 在VSCode中按Ctrl+Shift+P
3. 输入"Extensions: Install from VSIX"
4. 选择下载的VSIX文件安装

## 使用方法
- 命令面板输入"显示代码统计"查看数据
- 输入"导出代码统计"生成报告
- 状态栏实时显示输入统计
